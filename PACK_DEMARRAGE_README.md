## Utilisation du pack

### Import automatique

Le pack sera chargé automatiquement au premier lancement sur Netlify via le code d'initialisation dans `main.js`.

### Structure de chargement

```javascript
// Exemple de code d'import (à implémenter)
async function chargerPackDemarrage() {
    const response = await fetch('pack-demarrage.json');
    const pack = await response.json();

    // Charger dans localStorage/IndexedDB
    localStorage.setItem('infoCours', JSON.stringify(pack.infoCours));
    localStorage.setItem('groupeEtudiants', JSON.stringify(pack.groupeEtudiants));
    localStorage.setItem('grillesTemplates', JSON.stringify(pack.grillesTemplates));
    localStorage.setItem('echellesTemplates', JSON.stringify(pack.echellesTemplates));
    localStorage.setItem('productions', JSON.stringify(pack.productions));
    localStorage.setItem('cartouches_' + pack.grillesTemplates[0].id,
                        JSON.stringify(pack['cartouches_GRILLE1759243306842']));
    localStorage.setItem('evaluations', JSON.stringify(pack.evaluations));
    localStorage.setItem('presences', JSON.stringify(pack.presences));
    localStorage.setItem('calendrierComplet', JSON.stringify(pack.calendrierComplet));
    localStorage.setItem('seancesCompletes', JSON.stringify(pack.seancesCompletes));
    localStorage.setItem('modalitesEvaluation', JSON.stringify(pack.modalitesEvaluation));

    // Déclencher les calculs d'indices
    await calculerEtStockerIndicesCP();
    await calculerIndicesAssiduité();
}
```

## Notes techniques

### Format des données

- **Évaluations** : Notes sur échelle IDME (1.0-4.0), converties en % (0-100)
- **Présences** : Boolean `present` + nombre `heures` (0.0 ou 2.0)
- **Dates** : Format ISO 8601 (YYYY-MM-DD)

### Compatibilité

- **localStorage** : Tous les modules fonctionnent avec localStorage
- **IndexedDB** : Compatible avec architecture hybride Beta 91.1+
- **Import/Export** : Compatible avec système CC BY-NC-SA 4.0

MONITORAGE PEDAGOGIQUE - VERSION BETA 0.92
================================================================================

Une nouvelle version qui vous accompagne des premiers pas au partage

================================================================================
L'ESSENTIEL
================================================================================

Cette nouvelle version introduit **Primo**, votre assistant personnel qui vous
guide dans la decouverte de l'application, ainsi qu'un systeme de partage de
materiel pedagogique avec licences Creative Commons integrees.

================================================================================
NOUVEAUTE MAJEURE : PRIMO, VOTRE ASSISTANT DE DEMARRAGE
================================================================================

------------------------------------------------------------------------
Un accompagnement personnalise des la premiere utilisation
------------------------------------------------------------------------

Lorsque vous ouvrez l'application pour la premiere fois, **Primo vous accueille
automatiquement** et vous propose un tour guide en 4 modules :

MODULE 1 : Creer un groupe-cours (3 minutes)
   Primo vous pose quelques questions simples pour configurer :
   - Les informations de votre cours
   - Le calendrier du trimestre
   - Votre horaire hebdomadaire
   - Votre liste d'etudiants (import ou copier-coller)

   **Bonus** : Du materiel pedagogique de base est importe automatiquement
   (echelle IDME, grille d'evaluation, cartouches de retroaction)

MODULE 2 : Evaluer une production
   Primo vous guide directement vers le mode d'evaluation avec :
   - Navigation automatique vers les bons menus
   - Explication des etapes
   - Acces au materiel pedagogique

MODULE 3 : Explorer les diagnostics (bientot disponible)
   - Prevu pour une prochaine version
   - Comprendra une visite guidee des indicateurs A-C-P

MODULE 4 : Creer ma pratique de notation (8 minutes)
   Primo lance le Wizard qui vous aide a configurer :
   - Votre pratique de notation (sommative ou PAN-Maitrise)
   - Vos seuils d'interpretation
   - Votre echelle de performance
   - Votre grille de criteres

------------------------------------------------------------------------
Tutoriel interactif en 7 etapes
------------------------------------------------------------------------

Apres avoir complete le MODULE 1, Primo vous propose un **tutoriel interactif**
qui vous montre ou se trouvent les principales sections :

1. Tableau de bord - Vue d'ensemble de votre groupe
2. Etudiants - Gestion de votre liste
3. Presences - Saisie de l'assiduite
4. Evaluations - Creation et correction
5. Matériel - Vos ressources pedagogiques
6. Reglages - Configuration de l'application
7. Aide - Documentation complete

Chaque etape **met en evidence visuellement** la section concernee avec une
fleche et une explication claire.

------------------------------------------------------------------------
Section Aide enrichie et reorganisee
------------------------------------------------------------------------

La section Aide a ete completement restructuree pour faciliter l'apprentissage :

- **Introduction** : Concepts de base du monitorage pedagogique
- **Configuration initiale** : Guide detaille de votre premiere configuration
- **Utilisation hebdomadaire** : Workflows typiques (presences, evaluations)
- **Consultation et analyse** : Comprendre les diagnostics A-C-P
- **Reference** : FAQ et glossaire complets

**Nouveau** : Vous pouvez acceder directement a l'Aide depuis le modal Primo
pour consulter les guides detailles a votre rythme.

------------------------------------------------------------------------
Materiel pedagogique de demarrage
------------------------------------------------------------------------

Pour vous permettre de tester immediatement l'application, Beta 92 inclut :

- **Echelle IDME** : 5 niveaux (0, I, D, M, E) selon la taxonomie SOLO
- **Grille Global-5 FR-HOLIS** : 5 criteres d'analyse litteraire
  * Structure (15%)
  * Rigueur (20%)
  * Plausibilite (10%)
  * Nuance (25%)
  * Francais ecrit (30%)
- **Cartouches de retroaction** : 16 commentaires predéfinis pour la
  production "Carte mentale" (4 criteres × 4 niveaux)
- **Production exemple** : Artefact 3 - Carte mentale (type portfolio)
- **Liste de 10 etudiants fictifs** : Pour tester sans vos vraies donnees

Ce materiel est importe automatiquement lors du MODULE 1 de Primo.

================================================================================
PARTAGE DE MATERIEL PEDAGOGIQUE AVEC CREATIVE COMMONS
================================================================================

------------------------------------------------------------------------
Export de votre materiel avec attribution professionnelle
------------------------------------------------------------------------

Vous pouvez maintenant **exporter individuellement** :
- Vos grilles d'evaluation
- Vos echelles de performance
- Vos cartouches de retroaction
- Vos productions etudiantes
- Vos ensembles d'objectifs d'apprentissage

**Nouveaute** : Avant chaque exportation, un modal vous permet de specifier :
- Votre nom et prenom
- Votre discipline (ex: Litterature, Chimie, Philosophie)
- Votre institution (ex: Cegep de Drummondville)
- Votre courriel (optionnel - si vous souhaitez etre contacte)
- Votre site web (optionnel)
- Une courte description du materiel

**Protection de votre vie privee** : Vous choisissez exactement quelles
informations vous souhaitez partager en cochant/decochant les options.

------------------------------------------------------------------------
Import facilite du materiel de collegues
------------------------------------------------------------------------

Vos collegues peuvent maintenant **importer directement** votre materiel :
- Un seul fichier .json a telecharger
- Import en un clic depuis l'application
- Toutes les informations pedagogiques preservees
- **Attribution automatique** : Votre nom reste associe a votre travail
- Badge Creative Commons affiche lors de l'import

**Nouveaute** : Les boutons d'import/export apparaissent seulement quand
c'est pertinent (mode modification, pas en mode creation).

------------------------------------------------------------------------
Licence Creative Commons BY-NC-SA 4.0
------------------------------------------------------------------------

**Qu'est-ce que ca signifie pour vous ?**

La licence utilisee par defaut signifie que :
- Vos collegues peuvent **utiliser et adapter** votre materiel
- Votre **nom reste toujours associe** a votre travail (Attribution)
- Usage **educatif seulement** (pas d'utilisation commerciale)
- Les adaptations doivent aussi etre **partagees sous la meme licence**

**C'est du partage professionnel respectueux de votre expertise.**

Vous pouvez aussi choisir d'autres variantes de licences CC selon vos besoins
(CC BY, CC BY-SA, etc.).

================================================================================
AMELIORATIONS DE L'INTERFACE
================================================================================

------------------------------------------------------------------------
Navigation optimisee
------------------------------------------------------------------------

Les sections de materiel pedagogique ont ete renommees pour plus de clarte :
- Productions → **Productions etudiantes**
- Echelle de performance → **Echelles de performance**
- Retroactions → **Cartouches de retroaction**
- Objectifs d'apprentissage → **Ensembles d'objectifs**

------------------------------------------------------------------------
Visibilite contextuelle
------------------------------------------------------------------------

Les boutons d'action apparaissent maintenant seulement quand c'est logique :
- **Mode creation** : Boutons Exporter/Dupliquer caches
- **Mode modification** : Tous les boutons disponibles

Cela reduit la confusion et rend l'interface plus claire.

------------------------------------------------------------------------
Formulaires optimises
------------------------------------------------------------------------

Le formulaire de configuration du cours utilise maintenant **3 colonnes**
au lieu de 4, ce qui optimise l'espace vertical et reduit le defilement.

Nouveaux champs ajoutes :
- **Discipline** : Votre domaine d'enseignement
- **Institution** : Votre etablissement

Ces informations sont utilisees lors de l'export de materiel pedagogique.

CORRECTIONS IMPORTANTES
================================================================================

------------------------------------------------------------------------
Gestion des interventions RaI
------------------------------------------------------------------------

**Bug corrige** : Les interventions pedagogiques de niveau 2 et 3 peuvent
maintenant etre sauvegardees correctement avec la liste des participants.

Modal Creative Commons
------------------------------------------------------------------------

**Bug corrige** : Le modal de metadonnees Creative Commons apparait maintenant

**Technique** : Les fonctions asynchrones sont maintenant gerees avec

Smart Checkboxes
------------------------------------------------------------------------

Les checkboxes dans le modal d'export sont maintenant **intelligentes** :
- Cochees automatiquement si le champ contient une valeur

Cela evite de partager des informations manquantes par erreur.

POUR COMMENCER AVEC BETA 92
================================================================================

**Si vous etes un nouvel utilisateur** :

1. Ouvrez l'application - Primo vous accueillera automatiquement
2. Suivez le MODULE 1 pour configurer votre cours (3 minutes)
3. Completez le tutoriel interactif en 7 etapes
4. Consultez la section Aide pour approfondir vos connaissances
5. Testez l'evaluation avec les donnees de demonstration

**Si vous etes un utilisateur existant** :

1. Vos donnees sont preservees, rien ne change
2. Explorez la section Aide reorganisee
3. Testez le nouveau systeme d'export avec Creative Commons
4. Partagez votre materiel avec vos collegues en toute confiance

**Pour partager votre materiel** :

1. Allez dans Matériel → (Grilles/Echelles/Cartouches/Productions)
2. Cliquez sur "Exporter cette..." sous la ressource a partager
3. Remplissez le modal de metadonnees Creative Commons
4. Telechargez le fichier .json genere
5. Partagez-le avec vos collegues (courriel, Teams, OneDrive, etc.)

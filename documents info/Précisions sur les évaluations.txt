Il s'agit de la page «liste des ÉVALUATIONS». Éventuellement, nous ferons une page «liste des productions» pour bien distinguer l'une de l'autre (ex. une production avec ses stats : type, nombre de mots, objectifs, tâche, savoirs, savoir-faire, savoir-être, mode de réalisation individuel ou en équipe, etc), taxonomie (Bloom, SOLO, ELLAC), remise papier ou électronique, etc.

1. Une ligne
Ici, une "ligne" = 1 production évaluée produite par un élève (ex : "DA 2564910 - Artefact 1 - Stratégies de lecture")

2. Les types de productions
Ls types sont stockés dans les données de la page «Productions» dans les réglages. Ici, on retrouve  «artefact-portfolio» et «portfolio» dans les productions. Les autres types (examen, quiz, présentation, etc.) ne sont pas utilisés dans mon cours, mais je veux les offrir aux autres utilisateurs de l'application. C'est une bonne idée de définir ces types dans les productions (ex. : présentation, examen, etc). Il faudrait laisser une catégorie libre pour permettre à l'enseignant de créer des productions originales et innover)

3. Filtre "Évalué/Non évalué"
Bonne idée pour une production : "a au moins une évaluation" vs "aucune évaluation"

4. Filtre "Note globale"
Voir système de notation) : Si c'est une évaluation sommative traditionnelle, la note est un %. Si c'est une PAN, voir quel type de PAN (maitrise, spécification, dénotation)
Ma pratique de notation est une plage de notes non chiffrée (ex : IDME) 
La dénotation est plutôt de type échec/réussite

5. Les boutons d'action
Verrouiller : empêcher la modification de l'évaluation
Modifier : ouvrir la grille d'évaluation
Dupliquer : copier l'évaluation pour un autre élève
Supprimer : supprimer l'évaluation

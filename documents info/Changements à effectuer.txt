CHANGEMENTS À EFFECTUER
=======================

PHASE 1 - TEXTES VISIBLES (interface)
--------------------------------------
□ "Nouvelle évaluation" → "Nouvelle grille d'évaluation" 
□ "Modifier l'évaluation" → "Modifier la grille"
□ Label "Production étudiante :" → "Évaluation à appliquer :"
□ "-- Choisir une production --" → "-- Choisir une évaluation --"

PHASE 2 - VARIABLES JAVASCRIPT (code)
--------------------------------------
□ listeEvaluations → listeGrilles
□ ajouterEvaluation → ajouterGrille
□ sauvegarderEvaluation → sauvegarderGrille
□ afficherFormEvaluation → afficherFormGrille

PHASE 3 - IDs HTML (identifiants)
----------------------------------
□ evalTitre → grilleTitre
□ evalType → grilleType
□ evalPonderation → grillePonderation
□ evalCode → grilleCode

PHASE 4 - LOCALSTORAGE (données sauvegardées)
----------------------------------------------
□ localStorage 'listeEvaluations' → 'listeGrilles'
□ localStorage 'eval_XXX' → 'resultats_XXX'

9. FORMULES POSITIONNEMENT
────────────────────────────────────────────────────────────────────────────────
ACPE:
  position = (valeur - 0.30) / 0.70 * 100
  Mappe la plage 0.30-1.00 vers 0-100%

Patterns/RàI:
  position = positionZone + decalage + jitter
  decalage SOM: -2%  (à gauche)
  decalage PAN: +2%  (à droite)
  jitter: (Math.random() - 0.5) * 3.0 + constraint

Contrainte finale:
  positionFinale = Math.max(0, Math.min(position + jitter, 100))

11. DÉPANNAGE RAPIDE
────────────────────────────────────────────────────────────────────────────────
Points invisibles?
  → Vérifier afficherSom/afficherPan = true
  → Vérifier z-index: 10 ou plus
  → Vérifier position: relative sur parent

Gradient uni?
  → Vérifier syntax "linear-gradient(to right, ...)"
  → Vérifier codes couleur #RRGGBB
  → Vérifier pourcentages (0%, 25%, 50%, etc)

Animation ne marche pas?
  → Vérifier .barre-etudiant:hover { animation: float-subtle... }
  → Vérifier @keyframes float-subtle existe
  → Forcer refresh: Cmd+Maj+R

Couleurs différentes?
  → Tester sur autre navigateur
  → Vérifier codes héxa exacts
  → Checker l'overlay opacity
